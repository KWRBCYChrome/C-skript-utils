# skript essentials
## (NOTE: this is forked from my actual [skript-utils](https://github.com/KeyWhiteRBCY/skript-utils) repo.)
> a library of lightweight skripts for Paper servers that need it, this is for the most part commands. i will add stuff as i go and also by request
### commands:
| command         | arguments                      | description                                                                                                                                            | permission |
| --------------- | ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------- |
| setspawn        | none                           | set the location for the /spawn command                                                                                                                |            |
| spawn           | none                           | teleports you to the location set with /setspawn                                                                                                       |            |
| to              | player                         | teleport to arg-1                                                                                                                                      |            |
| bring           | player                         | bring arg-1 to you                                                                                                                                     |            |
| tor             | none                           | teleport to a random player                                                                                                                            |            |
| bringa          | none                           | bring every player to you                                                                                                                              |            |
| fly             | optional player                | toggle you or another player's flight                                                                                                                  |            |
| clearchat       | none                           | clear the chat                                                                                                                                         |            |
| sc or staffchat | text                           | chat with people with the permission                                                                                                                   |            |
| v or vanish     | none                           | hide you from the tablist and players                                                                                                                  |            |
| broadcast       | text                           | broadcast a server-wide message                                                                                                                        |            |
| telefrag        | player                         | teleport to and kill arg-1                                                                                                                             |            |
| report          | player and text                | report arg-1 with the reason being arg-2                                                                                                               |            |
| suggest         | text                           | suggest arg-1                                                                                                                                          |            |
| reportlist      | none                           | see the list of reporters, who they reported, the reason and when                                                                                      |            |
| suggestlist     | none                           | see the list of suggesters, what they suggested and when                                                                                               |            |
| clearlist       | optional text (report or sugg) | clears either both the suggestions list and reports list (no args) or specifically the reports (arg-1 = report) or the suggestions (arg-1 = sugg) list |            |
